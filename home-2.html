Process

From script to stage.

Phase 1

Read the Script

Identify the emotional beats: silence, reveal, confrontation, relief—then define the lighting vocabulary that supports them.

Phase 2

Design the Plot

Build focus architecture: angles, zones, and practical logic. Every fixture has a narrative job.

Phase 3

Program the Cues

Timing is writing. Fade curves and snap decisions become rhythm—clean programming, no surprises.

Phase 4

Tech Rehearsal

Refine focus, protect skin tones, eliminate spill, and lock the final cue stack with the director’s intent.
